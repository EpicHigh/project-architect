# Code of Conduct

This project follows the **Contributor Covenant v2.1**.

The full text is available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

If you experience or witness unacceptable behavior, please contact the project maintainer via GitHub Issues or the contact methods listed in the maintainer's profile.

## Scope

This Code of Conduct applies within all project spaces — issues, pull requests, discussions, and any other communication channels associated with this project.
