# Project Context — react-next-app

## Tech Stack

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Database:** PostgreSQL
- **ORM:** Prisma

## Directory Structure

- `app/` — Next.js app router pages and layouts
- `components/` — reusable React components
- `lib/` — shared utilities and helpers
- `prisma/` — database schema and migrations

## Key Commands

- **Dev:** `npm run dev`
- **Build:** `npm run build`
- **Test:** `npm test`
- **Lint:** `npm run lint`
