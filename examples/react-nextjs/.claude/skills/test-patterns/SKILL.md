# Test Patterns — react-next-app

## Framework

- **Test runner:** Jest
- **Libraries:** @testing-library/react, @testing-library/jest-dom

## Commands

- **Run all:** `npm test`
- **Run one:** `npx jest components/Button.test.tsx`
- **Coverage:** `npm run test:coverage`
